Installation
============

Requirements
------------

RadarSim requires Python 3.9+ and the following dependencies:

.. code-block:: bash

   pip install -r requirements.txt

Dependencies:

* numpy>=1.20.0
* scipy>=1.7.0
* numba>=0.56.0
* PyQt6>=6.0.0
* pyqtgraph>=0.13.0
* h5py>=3.7.0
* pyyaml>=6.0.0

Quick Installation
------------------

.. code-block:: bash

   git clone https://github.com/SpaceEngineerSS/RadarSim.git
   cd RadarSim
   pip install -r requirements.txt
   python run_gui.py
